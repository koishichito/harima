# はりま産学交流会 公式サイト リブランディング

企業と学生をつなぐ、地域密着型の産学連携プラットフォーム「はりま産学交流会」の公式サイトリブランディングプロジェクトです。

## プロジェクト概要

学生視点での分かりやすさと親しみやすさを重視し、UI・デザインを全面的に改善しました。

**デプロイ済サイト**: https://harima-sigma.vercel.app  
**現在の公式サイト**: https://www.sangaku.jp/

## 実装ページ

### メインページ
- **トップページ** (`index.html`) - 企業×学生プロジェクトコンテストの告知、次回イベント案内
- **交流会のご紹介** (`pages/about.html`) - 活動内容、沿革、関連組織、事務局情報
- **会員企業のご紹介** (`pages/members.html`) - 参加企業の紹介、業種別分類、参加メリット
- **入会申込** (`pages/admission.html`) - 入会フォーム、会費情報、注意事項
- **お問い合わせ** (`pages/contact.html`) - 問い合わせフォーム、FAQ、連絡先情報
- **これまでの歩み** (`pages/history.html`) - 30年の歴史、タイムライン、マイルストーン
- **連携組織** (`pages/partners.html`) - 研究発表機関、協力・後援機関、連携事例

## デザインシステム

### カラーパレット
- **プライマリカラー**: #003d7c (青)
- **セカンダリカラー**: #0052a3 (明るい青)
- **テキストカラー**: #333 (ダークグレー)
- **背景カラー**: #f8f9fa (ライトグレー)

### タイポグラフィ
- **H1**: 2.5rem (40px)
- **H2**: 2rem (32px)
- **H3**: 1.5rem (24px)
- **本文**: 16px
- **フォント**: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Hiragino Sans'

### レスポンシブ対応
- **デスクトップ**: 1024px以上 (3-4カラムグリッド)
- **タブレット**: 768px〜1023px (2カラムグリッド)
- **モバイル**: 767px以下 (1カラムグリッド)

## 主な改善ポイント

### 1. 学生視点での分かりやすさ
- 視覚的な情報整理 (カード、アイコン、タグ)
- 数値データの可視化 (統計情報、実績)
- タイムラインデザイン (歴史の可視化)
- FAQ (よくある質問)
- 平易な言葉遣い

### 2. 親しみやすさ
- アイコンの活用 (絵文字、アイコン)
- カラフルなデザイン (グラデーション、背景色)
- ホバーエフェクト (アニメーション)
- カードデザイン (情報の整理)
- 余白の活用 (圧迫感の軽減)

### 3. UI・デザイン改善
- 統一感のあるデザインシステム
- レスポンシブ対応 (全デバイス最適化)
- アクセシビリティ (コントラスト比、フォントサイズ)
- パフォーマンス (画像最適化、CSS分離)
- SEO対策 (メタタグ、構造化データ)

### 4. 導線の工夫
- 明確なCTA (次のアクションを促すボタン)
- パンくずリスト (現在位置の明示)
- フッターリンク (全ページへのアクセス)
- 内部リンク (関連ページへの誘導)
- アクティブ状態の表示 (現在のページを強調)

## 技術仕様

### HTML
- セマンティックHTML5
- アクセシビリティ対応 (ARIA属性)
- メタタグ最適化 (SEO対策)

### CSS
- モダンCSS (Flexbox, Grid)
- レスポンシブデザイン (メディアクエリ)
- トランジション・アニメーション
- カスタムプロパティ (CSS変数)

### JavaScript
- バニラJavaScript (依存ライブラリなし)
- フォームバリデーション
- モバイルメニュートグル
- スムーススクロール

## 開発環境

### ローカル開発
```bash
# リポジトリをクローン
git clone https://github.com/koishichito/harima.git

# ディレクトリに移動
cd harima

# ローカルサーバーを起動 (例: Python)
python3 -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```
